# Example dataset

This example dataset contains a subset of the samples in the [SPIDER dataset](https://zenodo.org/records/10159290) [1, 2] published under [CC-BY 4.0](https://creativecommons.org/licenses/by/4.0/legalcode)-license. The samples have been structured in a [BIDS](https://bids.neuroimaging.io)-like dataset structure.

See [../../examples/spider_to_bids.py](../../examples/spider_to_bids.py) on how to generate a BIDS-like compliant dataset from the source dataset for SPIDER.
